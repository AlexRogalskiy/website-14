Talks

  • Infinitely scalable chat

    Nelson

    Last month @tanzim introduced you to AWS Lambda ... this time we're going to show you how to build a chat/messenger app that scales to hundreds of thousands of messages per second without deploying a single Docker Container or manage a Database. Overview + Instead of using a (traditional) Node.js Web Server, we'll be using Lambda Functions. (Don't worry, its still node.js...) + Instead of a (traditional) Database, DynamoDB. + Instead of Pub/Sub and Socket.IO we will use AWS SNS + If there's time we can also show how to do Facebook/Twitter/Google Auth using AWS Cognito We will be writing the code for the app over the next couple of weekends. It will be similar to our previous demo app: https://github.com/dwyl/hapi-socketio-redis-chat-example in that there will be end-to-end tests and detailed documentation, but that's where the similarity ends! During the talk we will do a full stack/code/deployment walkthrough (aprox 15 mins) and then do AMA/Q&A (5 mins) We already have extensive practical notes on using AWS Lambda: https://github.com/dwyl/learn-aws-lambda ... these are going to be expanded into an end-to-end tutorial which everyone can follow. If you are curious about the future of web apps, give us a thumbs up.

  • What learning to code in 12 weeks really means

    Michael Lennox

    With Makers Academy currently hosting LNUG I thought it might be a great chance to give a talk to cover a few things about it including: a) what a bootcamp is, b) what they teach, c) what is actually meant by the concept of 'learning to code in 12 weeks', d) what it feels like as a student to push yourself to go from very little coding capacity at the start to being able to write, test and deploy applications 3/4 months later, e) the contrast between CS and bootcamp style education and why there is such a demand for alternate education sources in the current market I would be very interested in giving a talk of how Makers Academy has shaped my transition from a very different career path into a junior developer passionate about what I do. The talk would be rather technology agnostic, although node is one of the technologies I have had the pleasure of working with (and had to get to grips with very rapidly during a craft-a-thon), it would be much more based around trying to let people know about the trials and tribulations of the Makers Academy student journey. I'm Michael, a junior developer who's just finished here at Makers You can learn a little more about me at michaellennox.me/blog
